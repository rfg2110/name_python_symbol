sep (separator – разделитель) 
    пример:
        print('aa', 'bb', 'cc', sep='*')
    вывод:
        aa*bb*cc

Таким образом, необязательный параметр sep команды print() 
позволяет установить строку, с помощью которой будут разделены аргументы (если их несколько) при печати.


end
У команды print() есть параметр end, определяющий, что нужно добавить в конец вывода. По умолчанию параметр 
end равен символу перевода строки (\n).
Таким образом, параметр end определяет строку, которая будет добавлена после вывода всех аргументов команды print().
    пример:
        print('Привет', end='!')
    вывод:
        Привет!

        
    Таблица:
        Оператор:    Описание:
        +            сложение
        -	         вычитание
        *	         умножение
        /	         деление
